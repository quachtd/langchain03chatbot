# langchain03chatbot

A chatbot with:
- LangChain 03
- LangGraph
- GUI with Chainlit

### Install
python3 -m venv .venv\
source .venv/bin/activate\
pip install -r requirements.txt

#### OpenAI Key
create an file .env
```
OPENAI_API_KEY='your key here'
```
### Run
chainlit run chatbot2_chainlit.py -w
